2. Application Software Types

We classify application software based on its usage:

  • General Purpose: Software that can be used for many different tasks (e.g., Word Processors, Spreadsheets).
  • Special Purpose: Software designed for a specific task (e.g., Accounting software, Web Browsers).
  • Bespoke (Custom): Software written specifically for one customer's needs.
Example Task:
A hospital needs software to manage patient records. Should they use Generic or Bespoke software?
Justification: Bespoke is better because patient data security and specific hospital workflows are critical.
