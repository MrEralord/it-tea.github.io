Context

1. The Hardware Foundation

Before understanding software, recall the Von Neumann Architecture.

  • CPU: The brain (CU + ALU).
  • Memory Hierarchy: Registers → Cache → RAM → Storage.

2. System vs. Application Software

Click image to zoom

  • System Software: Controls hardware (e.g., OS, Drivers).
  • Application Software: Programs designed for user tasks.

3. Classifying Application Software

Application software is organized into four distinct categories:

1. Business Word Processing, Spreadsheets.
2. Graphics CAD, Video Editing.
3. Education Learning aids, Finance.
4. Comms VoIP, Browsers.

4. Bespoke vs. Off-The-Shelf (COTS)

Comparison between Custom and Ready-made software:

Dimension Custom (Bespoke) Off-The-Shelf (COTS)
Cost High initial cost. Lower initial cost.
Suitability Tailored exactly to needs. General features.
Time Slow development. Immediate.
Critical Risk: Bespoke software risks failure to deliver.
